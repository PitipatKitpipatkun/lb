📖 คู่มือ Backup & Restore
⬇ Export JSON — ข้อมูลครบถ้วน ใช้สำรองหรือ restore
⬇ Export CSV — เปิดด้วย Excel ได้ทันที เหมาะวิเคราะห์ข้อมูล
⬆ Import JSON — นำเข้าข้อมูลจากไฟล์ backup เดิม (upsert = เพิ่ม/แก้ไข ไม่ลบ)
⬆ Import CSV — นำเข้าจาก Excel/CSV สำหรับ นักศึกษา / อาจารย์ / ทักษะ
📂 Import ทั้งหมด — นำเข้าจากไฟล์ backup_all_*.json ที่ Export ไว้
